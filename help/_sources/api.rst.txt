Команды
=========================================
Описание REST API.

Предоставляемое API позволяет управлять приложением и получать сведения о статусе его работы. Управление происходит через отправку http запросов на TCP порт Лутники. Такие запросы называются командами и делятся на 2 типа: **общие** и **контрольные**.

К общим относятся такие команды, как получение статуса работы. Обычно они доступны всем и потому позволяют выполнить только то, что может понадобиться лишь пользователю.
К контрольным относятся команды позволяющие получать все сведения о работе Лутники и контролировать её работу. Обычно они доступны только администраторам.

Параметры доступа к API задаются в секции :option:`[Server]`. Команды имеют свой синтаксис.
