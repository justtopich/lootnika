Общие команды
=========================================
Команды для пользовательского использования.

Для выполнения этих команд клиент должен быть в списке :option:`QueryClients`.

.. contents:: :local:


GetInfo
----------------------------------------
Получение общих сведений о Лутнике.

.. code-block:: monte

	a=getinfo


Возращаемые поля:

.. list-table::
   :widths: 25 15 60
   :header-rows: 1

   * - Поле
     - Формат
     - Значение
   * - product
     - строка
     - Название сборки Лутники
   * - version
     - строка
     - Версия
   * - picker_type
     - строка
     - Используемы сборщик данных
   * - directory
     - строка
     - Рабочая папка
   * - service_name
     - строка
     - Название службы
   * - client_host
     - строка
     - Адрес клиента
   * - client_role
     - строка
     - Уровень привилегий клиента 
   * - pid
     - число
     - PID процесса Лутники
   * - pid_owner
     - строка
     - владелец процесса


GetStatus
----------------------------------------
Получение сведений о работе Лутники.

.. code-block:: monte

	a=getstatus


Возращаемые поля:

.. list-table::
   :widths: 25 15 60
   :header-rows: 1

   * - Поле
     - Формат
     - Значение
   * - uptime
     - строка
     - Время работы
   * - cpu
     - число
     - информация о CPU
   * - ram_total
     - число
     - общий размер памяти
   * - ram_percent
     - число
     - % занятой всей памяти
   * - ram_used
     - число
     - размер занятой памяти
   * - ram_free
     - число
     - доступный размер реальной памяти (без swap)
   * - ram_available
     - число
     - доступный размер всей памяти
   * - ram_lootnika_percent
     - число
     - % занятой памяти Лутникой
   * - swap_total
     - число
     - общий размер SWAP памяти
   * - swap_used
     - число
     - размер использованного SWAP
   * - swap_free
     - число
     - доступный размер SWAP
   * - swap_percent
     - число
     - % использованного SWAP


Подробности про статистику использования ресурсов смотрите в официальной документации `psutil
<https://psutil.readthedocs.io/>`_.
