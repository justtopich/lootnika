Синтаксис настроек
=========================================
Конфигурационный файл состоит из секций, в котором есть параметры относящейся к ней.
Имеет следующую структуру:

.. code-block:: cfg

	[Секция1]
	Параметр1=значение
	Параметр2 = значение

	[Секция2]
	Параметр1 = значение
	Параметр2=значение
	; комментарий
	# это тоже комментарий
	

Название каждой секции обозначено квадратными скобками (например, :option:`[server]`). Секции не должны повторяться и могут находится в любом порядке. Параметр относится к секции до тех пор, пока не будет объявлена новая. Значения параметров пишутся в одну строку.

.. tip::
	В параметрах при указании путей используйте обратные слешы :guilabel:`/` вместо :guilabel:`\\`
	Если в пути есть пробелы - укажите его в кавычках: ``"home/var load/lootnika"``

Файл должен быть в кодировке UTF-8.
